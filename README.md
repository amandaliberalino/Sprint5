# Sprint5
Repositório para gerenciar arquivos referente ao projeto da Sprint 5 - Ferramentas de engenharia de software.


- Inseridas as configurações do stremlit para compatibilidade com Render em .stremlit/config.toml
- Os dados dos veículos estão armazenados no diretório data/vehicles.csv
- A EDA básica para teste de funcionalidade de gráfico da biblioteca plotly-express estão em notebooks/EDA.ipynb
- A versão do python utilizada está armazenada no arquivo .python-version para o Render
- o código do aplicativo se encontra no arquivo app.py
- Todas as bibliotecas para alocação no Render estão no arquivo requirements.txt
